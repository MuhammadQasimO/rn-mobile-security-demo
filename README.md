# Mobile Security Demo (React Native)

> Work in progress. Reference implementation of the mobile security stack
> I shipped on a production fintech app.

## Goals

- Native SSL pinning on iOS (Swift) and Android (Kotlin) — written
  from scratch, not a wrapper around a third-party library.
- HMAC request signing with timestamp-based replay protection.
- Companion Node server that validates signatures end-to-end.

## Stack

`React Native` · `TypeScript` · `Swift` · `Kotlin` · `Node.js` · `Express` · `HMAC-SHA256`
